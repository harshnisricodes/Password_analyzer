Password Strength Analyzer

Project Description:
        The Password Strength Analyzer is a simple web-based application built using HTML, CSS, and JavaScript that helps users check the strength of their passwords in real-time. It analyzes the password based on different security rules such as length, uppercase letters, numbers, and special characters, and provides instant feedback.
        
Features :
•Real-time password strength check
•Visual strength indicator (Weak / Medium / Strong)
•Animated strength bar
•User-friendly interface
•Suggestions to improve password security.

Technologies Used:
HTML       - Structure of the web page
CSS        - Styling and UI design
JavaScript – Password validation logic.

How It Works:
•User enters a password in the input box
JavaScript checks:
•Length of password
•Presence of uppercase letters
•Presence of numbers
•Presence of special characters

Objective:
  To help users understand password security and encourage them to create strong and secure passwords.
